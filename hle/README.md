# Home Lab Everywhere

Expose Home Assistant and other homelab services to the internet via [HLE](https://hle.world) tunnels, with built-in SSO protection.

Manage all your tunnels, access rules, and settings directly from the HA sidebar â€” no terminal required.

![HLE Add-on panel](https://hle.world/assets/ha-addon-preview.png)

## Features

- One-click expose Home Assistant
- Add tunnels for any service on your network
- SSO allow-list management per tunnel
- Live tunnel status (RUNNING / STOPPED / FATAL)
- Per-tunnel start/stop controls

## Requirements

- Home Assistant OS or Supervised installation
- An HLE account and API key ([hle.world](https://hle.world))
